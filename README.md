### 🏆 Featured Projects

#### 🌐 [Java DPI Engine](https://github.com/methlygupta23/data-packet-inspection)
Multi-threaded Deep Packet Inspection engine in Java. Reads `.pcap` files, identifies the app/website behind each connection (including encrypted HTTPS via TLS SNI), and applies configurable blocking rules through a 4-stage concurrent pipeline (Reader → Load Balancer → Worker Pool → Writer).

#### 🧠 [Own AI — Vector DB & RAG from Scratch](https://github.com/methlygupta23/Own-AI)
A vector database and RAG system built in Java with zero third-party dependencies. Implements Brute Force, KD-Tree, and HNSW nearest-neighbor search, a REST API with a built-in web UI, and an end-to-end RAG pipeline integrating local Ollama for embeddings and generation.

#### 💻 [NexSphere — Full-Stack Web Platform](https://github.com/methlygupta23/nexsphere)
An enterprise-style application with MVC architecture, a normalized MySQL database, RESTful APIs, and secure bcrypt-based authentication.
